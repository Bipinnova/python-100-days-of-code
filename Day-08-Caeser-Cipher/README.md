# 🔐 Day 08 - Caesar Cipher | 100 Days of Python

## 📌 Project Overview

This project is part of my **100 Days of Python Challenge**.

The objective of this project is to build a **Caesar Cipher** application that allows users to encrypt and decrypt messages by shifting characters in the alphabet.

Although the algorithm is simple, this project reinforces important programming concepts such as functions, loops, string manipulation, modular arithmetic, and conditional logic. It also introduces one of the earliest encryption techniques, helping strengthen the understanding of basic cryptography concepts.
