# ✊📄✂️ Day 04 - Rock Paper Scissors | 100 Days of Python

## 📌 Project Overview

This project is part of my **100 Days of Python Challenge**.

The objective of this project is to build a console-based **Rock, Paper, Scissors** game where the user competes against the computer.

The game uses Python's **random module** to generate the computer's choice and **lists** to store and display ASCII art for each move. It also reinforces decision-making using conditional statements while handling user input safely.

Although the game is simple, it revisits several core Python concepts that are commonly used in real-world applications.
