# 🔐 Day 05 - Password Generator | 100 Days of Python

## 📌 Project Overview

This project is part of my **100 Days of Python Challenge**.

The objective of this project is to build a **Password Generator** that creates strong and randomized passwords based on user preferences.

The user specifies the number of letters, symbols, and numbers to include, and the program generates a secure password by randomly selecting characters and shuffling their order.

Although the project is simple, it reinforces several important Python concepts that are widely used in automation, scripting, backend development, and security-related applications.
