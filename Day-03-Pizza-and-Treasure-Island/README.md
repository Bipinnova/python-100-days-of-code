# 🏝🍕 Day 03 - Pizza & Treasure Island | Conditional Logic | 100 Days of Python

## 📌 Project Overview

This project is part of my **100 Days of Python Challenge**.

Day 03 focuses on strengthening **Python Conditional Logic** by building two interactive console applications:

### 🍕 Project 1 - Pizza Order

A console-based pizza ordering system that calculates the final bill based on:

- Pizza Size
- Pepperoni Selection
- Extra Cheese

The project demonstrates how conditional statements are used to calculate prices dynamically.

---

### 🏝 Project 2 - Treasure Island

A text-based adventure game where players make a series of decisions to find hidden treasure.

The game uses multiple conditional statements and nested decision-making to determine different outcomes based on user choices.

Although both projects are beginner-friendly, they reinforce one of the most important programming concepts—**Conditional Logic**, which is widely used in real-world software development.
