# 🐞 Day 13 - Python Debugging Lab | 100 Days of Python

## 📌 Project Overview

This project is part of my **100 Days of Python Challenge**.

Unlike previous days, the objective of Day 13 was not to build a new application but to **analyze, debug, and fix existing Python programs**.

The exercises focused on identifying logical errors, understanding program flow, and using debugging techniques to ensure the code behaves as expected.

The debugging tasks included:

- Leap Year Checker
- FizzBuzz
- List Mutation Exercise

These exercises reinforced the importance of debugging, code analysis, and logical reasoning—essential skills for every software engineer.

## 💡 Engineering Takeaway

Debugging is one of the most valuable skills in software development.

These skills are widely used in backend development for troubleshooting APIs, debugging business logic, analyzing production issues, and improving overall software reliability.
