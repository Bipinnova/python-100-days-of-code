# 🐢 Day 19 - Turtle Race Game using Python Turtle Graphics

> **Part of my #100DaysOfCode Journey**

This project is a part of my journey to **revisit Python fundamentals** by building practical projects and strengthening core programming concepts.

For Day 19, I built a **Turtle Race Game** using Python's **Turtle Graphics** module. The application simulates a race between multiple turtles, where each turtle moves forward by a random distance until one reaches the finish line. Before the race begins, the user places a bet by selecting the color of the turtle they think will win.

This project provided a fun way to reinforce object-oriented programming, event handling, loops, randomness, and working with multiple objects simultaneously.

# 🎮 How to Play

1. Run the application.
2. Enter the color of the turtle you think will win.
3. Watch the turtles race across the screen.
4. The game announces whether your prediction was correct.

# 📸 Project Output

The program displays six colorful turtles lined up at the starting position.

Each turtle moves forward by a random distance until one crosses the finish line, after which the winner is announced based on the user's selected color.
