# 🐍 Day 20 - Snake Game using Python Turtle Graphics

> **Part of my #100DaysOfCode Journey**

This project is part of my journey to **revisit Python fundamentals** by building practical projects while strengthening object-oriented programming and software engineering concepts.

For Day 20, I built the classic **Snake Game** using Python's **Turtle Graphics** module. The game includes snake movement, food generation, score tracking, collision detection, and game-over conditions.

Rather than focusing only on game development, this project helped reinforce important programming concepts such as object-oriented design, modular programming, state management, and clean project organization.

# 🎮 Game Features

- Control the snake using the arrow keys.
- Eat food to increase your score.
- The snake grows after eating food.
- Avoid colliding with the wall.
- Avoid colliding with your own tail.
- Game Over is displayed when a collision occurs.

---

# 📸 Project Output

The game starts with a three-segment snake that moves continuously.

As the snake eats food, it grows longer while the score updates. The game ends when the snake collides with the wall or its own body.

This project demonstrates how larger applications can be built by combining multiple independent components.

The same design principles are widely used in backend development, where different services and modules collaborate to build scalable software systems.
